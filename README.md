## Featured projects

#### [PublicTradeBot](https://github.com/Le-skal/PublicTradeBot)
Automated trading system combining XGBoost predictions with NLP sentiment analysis on French financial news (L'Agefi RSS + CamemBERT). Trades BTC, ETH, and French stocks daily via GitHub Actions, with Google Sheets as a lightweight data store.

#### [Green-AI-Scanner](https://github.com/Le-skal/Green-AI-Scanner)
Full-stack platform that aggregates responses from 4 AI engines (Gemini, Mistral, HuggingFace, Cohere) and compares them on relevance (BM25), semantic similarity (TF-IDF), RGPD compliance, and carbon footprint. Built with React, Express, and MongoDB.

#### [datakit](https://github.com/Le-skal/datakit)
OOP Python library for managing ML datasets. Handles image and audio data with lazy/eager loading, batch creation, and a preprocessing pipeline (center crop, mel spectrogram, resampling, etc.). Built as a final project at the University of Groningen.
